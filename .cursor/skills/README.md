# Agent Skills (Cursor)

Skills do Cursor Agent: um subdiretorio por skill com **`SKILL.md`**.

**Neste repo**, as skills padrao estao em **`.claude/skills/`** (Cursor carrega essa pasta por compatibilidade). Evite duplicar; adicione skills novas la ou aqui conforme o time.

Cursor tambem pode carregar skills de `.agents/skills/` e `.codex/skills/`.

Ver [Agent Skills | Cursor Docs](https://cursor.com/docs/context/skills).

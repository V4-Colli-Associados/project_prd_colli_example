# Agent Skills (formato Codex / interoperavel)

Alguns fluxos do **OpenAI Codex** e do **Cursor** procuram skills em **`.agents/skills/`** (além de `.cursor/skills/` e paths sob `~`).

Estrutura: `<nome-da-skill>/SKILL.md` (+ pastas opcionais).

Ver `docs/08-AI-TOOL-CONFIG.md` e [Agent Skills — Codex](https://developers.openai.com/codex/skills/).

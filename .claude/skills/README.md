# Skills do Claude Code (projeto)

Cada skill e um diretorio com **`SKILL.md`** (e opcionalmente `scripts/`, `references/`, `assets/`).

Metodologia e guias longos do treinamento continuam em `docs/guides/` e `docs/templates/`.

Documentacao: [Claude directory — skills](https://code.claude.com/docs/en/claude-directory#ce-skills)

## Skills padrao deste starter

| Pasta | Uso |
|-------|-----|
| `sabatina-prd/` | Perguntas antes de executar; PRD final com decisao skill vs interface |
| `onboarding-vibe-coding/` | Conceitos basicos e como comecar |
| `organizar-temp-repositorio/` | Mover lixo `.md` (direcionamento, relatorios de sessao) para `temp/` |

O **Cursor** tambem descobre skills em `.claude/skills/` (compatibilidade). Nao e obrigatorio duplicar em `.cursor/skills/`.
